# Contributing

Merci de contribuer à `awesome-french-robotics`.

## Règles de sélection

- Le sujet doit être directement lié à la robotique, à la mécatronique, à l'automatique, à la perception robotique ou aux systèmes embarqués associés.
- La ressource doit avoir un lien explicite avec la France.
- Les liens officiels sont à privilégier.
- Les descriptions doivent être factuelles, courtes et non promotionnelles.
- Chaque entrée doit respecter le format `- [Nom](URL) - Description.`.
- Les entrées doivent être triées alphabétiquement dans la section concernée.

## Ce qui est généralement refusé

- Les entreprises purement IA ou purement software sans activité robotique claire.
- Les liens temporaires, cassés, redirigeant mal ou dépendant d'un suivi marketing.
- Les doublons.
- Les descriptions longues, vagues ou marketing.

## Avant d'ouvrir une PR

- Vérifiez que l'entrée n'est pas déjà présente.
- Vérifiez que l'URL répond bien.
- Placez l'entrée dans la section et la sous-section les plus pertinentes.
- Gardez le scope France-only.

## Pull Requests

- Une PR ciblée vaut mieux qu'une PR mélangeant beaucoup de changements non liés.
- Si vous ajoutez plusieurs entrées, indiquez brièvement la logique de sélection et les sources utilisées.
